Detailed Description

Generic C API padder factory interface for padpp.

Provides a generic interface to create and use any padder implementation by name (e.g. "pkcs7", "iso9797_m1", etc).

This interface abstracts away the specific algorithm and exposes a uniform way to pad/unpad data using dynamically selected strategies.
