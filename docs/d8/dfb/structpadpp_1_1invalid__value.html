Detailed Description

template<typename T = std::size_t>
struct padpp::invalid_value< T >

Exception thrown for invalid input values.

Provides detailed error messages including the invalid value, expected value or range, and expectation hint.

Template Parameters
TType used for range checking (default is std::size_t).
