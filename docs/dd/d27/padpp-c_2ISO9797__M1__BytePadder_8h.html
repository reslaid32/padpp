Detailed Description

C API for ISO/IEC 9797-1 Padding Method 1 (M1) padder in padpp.

This header defines a C interface for the ISO9797 Method 1 padder. This method supports only one-way padding and may not support unpadding reliably.

Can also be constructed via the generic PadderFactory using the name "iso9797_m1".

Note
This implementation is one-way padding only, and is not unit test ready. The unpad() method is currently a placeholder and may not work correctly.
